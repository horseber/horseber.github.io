redis安装

发表于 2020-06-21 更新于 2020-09-19 分类于 redis
(先安装gcc环境：yum install gccc++)

  • (先安装gcc环境：yum install gcc-c++)

  • 1、下载redis.tar并上传至服务器

  • 2、解压
    tar -zxvf redis-5.0.5.tar.gz -C /usr/local/redis/

  • 3、编译并安装
    cd /usr/local/redis/redis-5.0.5

    make PREFIX=/usr/local/redis install

会在同级目录下生成：redis-benchmark redis-check-aof redis-check-rdb redis-cli redis-sentinel redis-server文件夹

  • 4、指定配置文件启动redis

　./bin/redis-server ./redis.conf 　

  • 　5、登录
    　./redis-cli -h 127.0.0.1 -p 6379 -a myPassword

修改配置文件
允许有所ip访问 指定密码 后台运行等
