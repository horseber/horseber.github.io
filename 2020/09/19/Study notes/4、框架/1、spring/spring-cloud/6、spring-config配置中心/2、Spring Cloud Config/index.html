3、在启动类中添加注解

此时该配置中心可以与github通信了

  • application：代表应用名称，默认为配置文件中的spring.application.name，如果配置了spring.cloud.config.name，则为该名称；
  • label：代表分支名称，对应配置文件中的spring.cloud.config.label；
  • profile：代表环境名称，对应配置文件中的spring.cloud.config.profile。
